IDE

Replit 與 localhost 啟動與編輯網頁過程比較:

Replit 需要連網才能使用, localhost 可以在無網路時編輯已經 clone 到近端的倉儲內容, 並進行多次提交, 待有網路連線時再 push 到 Github.

Replit 可以在手機上使用.

Replit 在網頁上提供的 Tools 較為直覺, 且同一 Tool 可以在多個 Tab 中開啟使用

費用:

Replit: https://replit.com/pricing: Basic workspace

localhost: 自有實體或虛擬主機: 電費約 5000 NTD/yr
