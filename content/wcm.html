目前已知 cmsite 在 codespaces 環境中的單頁編輯會出現 bug, 必須手動進入單頁編輯模式.

必須將 cmsimde 中的 def correct_url() 中的 url 改為:

# 根據 https://stackoverflow.com/questions/15093593/request-path-and-url-for-dont-match-up-in-flask-under-mod-wsgi 中的說明修改
# 原先的 url = request.url 在 replit 與 gitpod 可行
# 但是在 codespaces 無法取得 wsgi 伺服器中正確的隨機網址
url = request.script_root + request.path
